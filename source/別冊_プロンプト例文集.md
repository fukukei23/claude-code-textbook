## 便利ワンライン

| 用途 | プロンプト |
|---|---|
| 短くして | 「3行以内で」 |
| 日本語で | 「説明はすべて日本語で」 |
| 急いで | 「高速モードで、簡易版でいいから」 |
| 教えて | 「なぜこうなっているか説明して」 |
| 検証して | 「このコードの動作を確認して」 |
| 比較して | 「AとBの長所・短所教えて」 |
