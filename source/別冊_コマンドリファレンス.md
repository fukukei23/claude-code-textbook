# 別冊 コマンドリファレンス — よく使うコマンド一覧

> Claude Codeで使うコマンドを一覧化。必要な時にすぐ引けるように。

---

## Claude Code CLI コマンド

### 起動・終了

| コマンド | 説明 |
|---|---|
| `claude` | 対話モードで起動 |
| `claude "質問内容"` | 質問を1回だけ実行して終了 |
| `claude update` | Claude Codeを最新版に更新 |
| `claude --version` | バージョン確認 |
| `/exit` | セッション終了（対話モード内） |

### セッション内スラッシュコマンド

| コマンド | 説明 |
|---|---|
| `/help` | ヘルプ表示 |
| `/clear` | 会話履歴をクリア（コンテキストリセット） |
| `/compact` | 会話を要約してコンテキストを節約 |
| `/model` | 使用モデルを変更 |
| `/context` | 現在のコンテキスト使用量を表示 |
| `/cost` | 現在のセッションの料金を表示 |
| `/permissions` | 権限設定を確認 |
| `/memory` | メモリファイルを確認 |
| `/tasks` | バックグラウンドタスク一覧 |
| `/review` | コードレビューを実行 |
| `/fast` | 高速モード切替 |
| `/undo` | 直前のファイル変更を取り消し |

### 起動オプション

| オプション | 説明 |
|---|---|
| `--model NAME` | モデルを指定して起動 |
| `--allowedTools` | 使用可能なツールを制限 |
| `--print` | 出力のみ表示（パイプ用） |
| `--output-format json` | JSON形式で出力 |

## Git基本コマンド

| コマンド | 説明 |
|---|---|
| `git init` | 新規リポジトリ作成 |
| `git add -A` | 全変更をステージング |
| `git commit -m "メッセージ"` | コミット |
| `git push` | リモートにプッシュ |
| `git pull` | リモートから取得 |
| `git status` | 変更状況確認 |
| `git log --oneline -10` | 直近10件のコミット表示 |
| `git diff` | 変更差分確認 |

## WSL2 コマンド

| コマンド | 説明 |
|---|---|
| `wsl --install` | WSL2インストール（PowerShell） |
| `wsl --list --verbose` | インストール済みディストリビューション一覧 |
| `wsl --set-default Ubuntu` | デフォルトをUbuntuに設定 |
| `wsl --update` | WSL2更新 |

## 便利なLinuxコマンド

| コマンド | 説明 |
|---|---|
| `ls -la` | ファイル一覧（隠しファイル含む） |
| `cd ~/projects` | ディレクトリ移動 |
| `mkdir -p dir/sub` | ディレクトリ作成（中間も含む） |
| `cat file.txt` | ファイル内容表示 |
| `grep "keyword" file` | ファイル内検索 |
| `find . -name "*.py"` | ファイル検索 |
| `python3 script.py` | Pythonスクリプト実行 |
| `npm install -g package` | Node.jsパッケージのグローバルインストール |
