AI到底是什么？

AI = 言葉を理解して答えるコンピュータ

AI（人口知能）は、大量のテキストを読んで「こういう時にはこう答える」を學習したコンピュータープログラムです。

ポイント 説明
文章を理解する 日本語の質問を理解して答えを生成
知識が豊富 学習データ量大，所以知道很多东西
嘘をつくことがある 自信满满地说错误的话，叫「幻觉」（ハルシネーション）

⚠️ 重要: AIの答えを信じる前に、自分でも確かめよう！

AIとClaude Codeの関係

AI（大脑）= Claude Code（工具）= 程序员帮你写代码
  • AI: 答えを考える部分
  • Claude Code: AIと电脑の间にいて、帮助你写代码的工具

Claude Code 超まとめ

一言でいうと

Claude Codeは、AIに帮你写代码的工具。コマンドを入力すると、AIが代わりにコードを書いてくれます。

使えるコマンド

コマンド 意味
claude 対話モードで起動
claude "質問" 1回だけ質問して終了
/help ヘルプを表示
/clear 会話をリセット
/exit 終了

便利機能

機能 有什么用
スキル 자주使う指示を保存해서自動化
Memory セッションをまたいで覚えておく
フック 特定のタイミングで自动実行
MCP 外部サービスを连接して機能を扩展

プロンプトのコツ

良いプロンプトの3原則

原則 例
具体的に 「ファイルを创建して」→ 「todo-appというフォルダにindex.htmlを作成して」
結果を指定 「表形式で出して」「日本語で答えて」
段階的に 复杂な任务是先把大框架说明白

よく使うパターン

✅ 良い例: 「Pythonで hello.py を作成。中身は 'Hello, World!' と表示する」
❌ 悪い例: 「作って」

GitHub 超まとめ

一言でいうと

GitHubは、コードをクラウドに保存できるサービス。消えない・変更履歴が残る・他の人と共同できる。

基本の流れ

作業 → git add（選択）→ git commit（記録）→ git push（アップロード）
コマンド 意味
git clone URL 初回にリポジトリをコピー
git add ファイル コミット対象を選ぶ
git commit -m "説明" 変更を記録
git push クラウドに上げる
git pull 最新版を取得

❌ GitHubにAPIキーをpush → 世界中に公開 → 巨额請求

お金の知識

  • AIは「文字量」で料金が決まる
  • /context で今のコストを確認できる
  • 長いセッションは `$（ 돈）」が沙汰、高額になる

知識管理（SSOT）の概念

SSOT = Single Source of Truth

「情報が散らばるとややこしいので、正しい情報を1箇所にまとめる」という考え方。

❌ 困ったとき
   - メモAには○○と書いてある
   - メモBには××と書いてある
   - どちらが正しいか分からない！

✅ SSOTなら
   - 正しい答えはここにしかない
   - 迷ったらここを見ればいい
